# ナインタイル解説中級編：お題の8分類

ナインタイルのお題は8種類に分類できます。

評価軸は２つあり、以下です。

* 軸①寒暖色の枚数差(暖-寒)
  * 取り得る値：3,1,-1,-3（4種類）

* 軸②登場する模様の数
  * 取り得る値：5,6（２種類）

軸①（４種類）×軸②（２種類）＝８種類





## 軸１について：寒暖色の枚数差（暖-寒）のメカニズム

ナインタイルの全9枚のタイルは、その構成により以下の3種類に分類されます（初級編復習）。

1. **純暖色タイル（3枚）：** 両面とも「暖色グループ」のタイル。どう裏返しても、盤面に「暖色」を1つ増やします。
2. **純寒色タイル（3枚）：** 両面とも「寒色グループ」のタイル。どう裏返しても、盤面に「寒色」を1つ増やします。
3. **クロスエッジタイル（3枚）：** 片面が暖色、片面が寒色のタイル。この3枚の裏返し方によってのみ、盤面全体の「暖色・寒色」のバランスが変化します。

この構成を前提に、取り得る「暖色数 - 寒色数」の値を計算すると、以下の4通りになります。

- **ケース①（暖色最大）：** クロスエッジ3枚をすべて「暖色」にした場合
  - 暖色：3枚(純) + 3枚(クロス) = **6枚**
  - 寒色：3枚(純) + 0枚(クロス) = **3枚**
  - 差：$6 - 3 = \mathbf{3}$
- **ケース②（暖色多め）：** クロスエッジ2枚を「暖色」、1枚を「寒色」にした場合
  - 暖色：3枚(純) + 2枚(クロス) = **5枚**
  - 寒色：3枚(純) + 1枚(クロス) = **4枚**
  - 差：$5 - 4 = \mathbf{1}$
- **ケース③（寒色多め）：** クロスエッジ1枚を「暖色」、2枚を「寒色」にした場合
  - 暖色：3枚(純) + 1枚(クロス) = **4枚**
  - 寒色：3枚(純) + 2枚(クロス) = **5枚**
  - 差：$4 - 5 = \mathbf{-1}$
- **ケース④（寒色最大）：** クロスエッジ3枚をすべて「寒色」にした場合
  - 暖色：3枚(純) + 0枚(クロス) = **3枚**
  - 寒色：3枚(純) + 3枚(クロス) = **6枚**
  - 差：$3 - 6 = \mathbf{-3}$

**結論：**

軸１（暖色数 - 寒色数）が取り得る値は、**「3, 1, -1, -3」** の4種類です。



## 軸２について：お題の制約と、模様の種類の数

ナインタイルで出題される問題は、
9つのタイルで作れるすべての状態が候補になっているわけではありません。
「同一の模様（色）は0~2個」になっています。
つまり、一つのお題に「桜が３つ」「ブロックが３つ」ある状態はありえないのです。
これを踏まえると、最低でも一つのお題に模様が5種類登場することがわかります。

---

**証明（読みとばして良い）**

背理法を用いて証明します。

仮に、登場する模様の種類が「4種類以下」であると仮定します。

制約により、1つの模様につき盤面に配置できるのは最大で「2個」までです。

このとき、盤面に配置できるタイルの総数の最大値は、

$$4 \text{（種類）} \times 2 \text{（個/種類）} = 8 \text{（個）}$$

となります。

しかし、ナインタイルの盤面を完成させるには「9個」のタイルが必要です。

最大で8個までしか置けないため、9個目のマスを埋めることができず、矛盾が生じます。

したがって、9個のマスをすべて埋めるためには、4種類では不足であり、**最低でも「5種類」の模様が登場しなければならない**ことが証明されました。

---

したがって、模様の種類は最低で5、最高で6(そもそも模様は6個だから)です。



# コラム：ナインタイルの問題の数

- **同一模様２枚制約ありの場合の数：**

  **1,587,600 通り**

  （上記のうち、「同じ模様は最大2個まで」という製品版のルールを満たすものの数）

- **実際に商品のナインタイルに含まれるお題の数：**

  **30 通り**

**考察：**

物理的に作成可能な150万通り以上の有効な盤面の中から、さらに厳選された30問だけがお題カードとして採用されています。これにより、簡単すぎず難しすぎない、絶妙なバランスの問題が提供されていることがわかります。
